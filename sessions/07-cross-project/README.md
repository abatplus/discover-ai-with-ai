# Session 7 – Cross-Project Learning (Optional)

> **Practical tip:** Claude can look at patterns from other projects and transfer them. You're not limited to a single codebase.

> **Important:** Start a fresh Claude Code session for this one. A new context keeps Claude focused.

**Goal:** Claude can transfer knowledge between projects.

**Demo:**

> **Prompt** – copy the following into Claude Code:
```
Look at the error handling in [path to another project on your machine].
How are errors handled there?
Transfer that pattern to our Task API project.
Explain what you're adopting and why.
```

**Your turn:** Show Claude a reference project you like. Have it transfer a pattern, architecture decision, or approach to the micro-project.

**Tips:**
- Claude can look at projects in other languages and translate concepts
- You can also say: "In project X we do it this way – adopt that here"

**What you learned:**
- Claude understands patterns across projects
- Best practices from one project can be transferred to another
- You're not limited to a single project
