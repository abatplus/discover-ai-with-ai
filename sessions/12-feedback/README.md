# Session 12 – Feedback

**Goal:** Reflect on your experience and give feedback so the learning path can be improved.

This is not a demo – it's a short conversation with Claude that helps you structure your feedback.

**How to start:**

> **Prompt** – copy the following into Claude Code:
```
I want to give my feedback now.
Read the file sessions/12-feedback/feedback-prompt.md
and start the interview.
```

**Topics to think about:**
- What surprised or excited you the most?
- What didn't work well or was unclear?
- What topics or use cases are missing?
- Would you use Claude Code in your daily work?
